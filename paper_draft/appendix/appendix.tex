\section{Reproducibility Details}
\label{app:repro}

\para{Environment.} All experiments were run with Python $3.12.8$, NumPy $2.5.0$,
scikit-learn $1.9.0$, SciPy $1.18.0$, and PennyLane $0.45.0$ using the
\texttt{lightning.qubit} statevector simulator on CPU. Seeds are fixed
(\texttt{numpy}/\texttt{python} $=42$; dataset seeds $0$/$1$/$2$). All models share
the same $10$ repeated stratified $70/30$ splits so that quantum-vs-classical
comparisons are paired.

\para{Per-seed paired statistics (Experiment~A).} \Tabref{tab:app_paired} reports
the paired $t$-test of the projected quantum kernel against the best classical
model for every (qubit count, seed) configuration. All $9$ configurations are
significant at $p<10^{-3}$, with effect sizes $d$ ranging from $1.9$ to $12.1$.
